# How to compile
- Enter the following command in this folder (host PC).
	> make push
: you can get object files (*.o) and target file (20171640) 
and target file transfers to the board (/data/local/tmp)

# How to run
- insmod for device driver must precedes. 
1) Go to the /data/local/tmp directory (board).
2) Run target file by entering the following command.
	> ./20171640
